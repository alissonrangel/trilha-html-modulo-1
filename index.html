História dos computadores

(voltar)

1a geração: década de 40

Com computadores gigantescos no tamanho e servindo para gerar cálculos complexos, que seria usado pelos Eua e Alemanha para a guerra, pois foram criados durante a 2a Guerra Mundial. Computador Eniac e o Colossus (Alemanha). Nesse tempo surgiu o termo bug, que significa inseto, pois os insetos pousavam nos equipamentos delicados e terminavam dando problemas nas máquinas.

2a Geração: 1959 - 1965

Substituição das válvulas e relés por transistores. Assim houve uma diminuição do tamanho dos computadores.

3a Geração: 1965 - 1970

Com circuitos integrados, deixando os computadores menores, mais portáteis e com maior rapidez de processamento. Com o início do uso de sistemas operacionais mais modernos.

4a Geração: 1971 - ...

Criação dos microprocessadores, computadores pessoais. Mais populares e acessíveis. Marcada também pela criação do mouse com seu uso em um ambiente gráfico.

É caracterizada por um aperfeiçoamento da tecnologia já existente, proporcionando uma otimização da máquina para os problemas do usuário, maior grau de miniaturização, confiabilidade e velocidade maior, já da ordem de nanosegundos (bilionésima parte do segundo).

Internet - como surgiu?

Pela necessidade de comunicação entre locais distantes. Começou com os telégrafos com código morse. Em 1958, foi feito um canal submarino ligando locais pelo oceano. Durante a Guerra Fria, acabou levando a grandes desenvolvimentos na tecnologia de computadores e de comunicação. Surgimento da DARPA. Surgimento da ARPANET, início da internet.

O que são Clients?

(voltar)

Na programação web, client significa o lado da comunicação entre client e server. Ele faz a requisição pelo browser para um server remoto. Este server remoto é o local que contém as informações que o client necessita. O client pode acessar a rede através de vários dispositivos como pcs, tablets, laptops ou smartphones. Eles acessam a internet atráves dos navegadores, ou por aplicativos que acessam os servers sem usar navegadores. Os navegadores também são chamados de browsers. Os mais comuns atualmente são o Chrome, o Firefox, o Safari e o Edge. Os navegadores, no lado client, receberão dos servers as aplicações web que ficam hospedadas neles. Javascript é uma linguagem de programação que roda do lado client.

O que são Servers?

(voltar)

O server é o lado da comunicação client - server que aguarda uma requisição do client para poder enviar uma resposta. A sua resposta ocorre através do envio de arquivos HTML, CSS, Javascript, imagens, etc. Ele pode estar com acesso direto a um servidor de banco de dados, aonde ele pega informações para enviar ao client, com base na requisição desse client. Existem servidores de: arquivos, segurança, streaming, e-mail e web. Eles podem estar em grandes datacenters como no quarto de uma casa. Existem sistemas operacionais próprios para servidores, como: Windows Server e servidores baseados no Linux. Existem linguagens de programações usadas no servidor, como: php, java, ruby, javascript e C#.

Como é uma estrutura básica de um HTML?

(voltar)

Uma página HTML é formada por tags que fazem marcações no arquivo, cada uma tendo sua finalidade. A primeira tag que existe é a html, que envolve todo o código. Dentro dela tem as tags head e body. Dentro da delimitação da tag head, existe outras tags que se utiliza, elas são tags de configurações do código html.

O elemento HTML head providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
Exemplos de tags existentes dentro da head são: meta, title, link, style. HTML head Tag Na tag body é que vai existir o que de fato vai aparecer para o usuário, nela existe dezenas de tags: tags de formatação, tags de estruturação do código, âncoras (são links para dentro ou para fora do documento), tags de formulário, tags de tabelas, etc. Cada tag possui um conjunto de atributos que se pode usar. Entendendo a diferença entre listas ordenadas e não ordenadas

(voltar)

Existem dois tipos de listas no html, a lista ordenada criada com a tag "ol" e a lista não ordenada criada com a tag "ul". A tag ordenada, vai ser usada para listar itens colocando uma numeração antes do item. Esta numeração pode ser com números cardinais, numeração romana e numeração alfabética. 

        <ol>
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ol>
  1. Item 1
  2. Item 2
  3. Item 3
        <ol type="I">
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ol>
  1. Item 1
  2. Item 2
  3. Item 3
        <ol type="A">
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ol>
  1. Item 1
  2. Item 2
  3. Item 3
A tag não ordenada cria a lista de itens com um marcador para cada item da lista. Este marcador pode ser um disco, um círculo ou um quadrado, por exemplo. 
        <ul type="disc">
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ul>
  • Item 1
  • Item 2
  • Item 3
        <ul type="circle">
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ul>
  • Item 1
  • Item 2
  • Item 3
        <ul type="square">
          <li>Item 1</li>
          <li>Item 2</li>
          <li>Item 3</li>
        </ul>
  • Item 1
  • Item 2
  • Item 3
